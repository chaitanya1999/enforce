Welcome to EnForce IDE

developed by Chaitanya V.

This online SF IDE acts as a VSCode powered Developer Console V2 allowing LWC development.

Features -

  • Connect to multiple orgs at once
  • Facilitates Apex, Aura, LWC, VF development and deployment across multiple orgs at once (using tooling API)
  • Allows code comparison across multiple orgs (right click on tab)
  • Allows to execute SOQL queries on any connected org
  • Allows to execute Anonymous Apex on any connected org


How to use :


  1. Enable CORS on your org. Go to Setup > CORS > Add this website domain to CORS whitelist.
  2. Tick "Enable CORS for OAuth endpoints".
  3. Go to Org Manager tab, click Add + icon.
  4. For authentication using username, password ( + security token ) choose Auth Mode = SOAP Login and enter the details.
  5. For authentication using accessToken / session ID choose Auth Mode = Access Token. To obtain Session Id from an existing logged in session follow next step.
  6. Run the below anonymous apex script in developer console to obtain Session ID and Instance URL

        String sessionId = userInfo.getSessionId();
    System.debug('Session id '+ UserInfo.getOrganizationId() + UserInfo.getSessionId().substring(15));
    System.debug(URL.getOrgDomainURL().toExternalForm());
  7. Test authentication via authenticate button

Known Issues -

  • Some issues with keyboard shortcuts due to browser default behavior
  • Code editor : tabs drag and drop

Keyboard Shortcuts - Code Editor

  • Ctrl + P = Switch to open class/component input field
  • Ctrl + S = Save code
  • Ctrl + [PgUp / Tab] = Cycle Tabs Forward
  • Ctrl + [PgDown / Shift+Tab] = Cycle Tabs Backwards
  • Ctrl + Shift + [PgDown/PgUp] = Reorder tabs left right
